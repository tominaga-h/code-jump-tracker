# Code Jump Tracker

VS Code でのコードジャンプ履歴を視覚的に管理する拡張機能。

## 機能

- **自動記録**: 「定義へ移動」やファイル間ジャンプを自動で履歴に記録
- **手動記録**: 右クリックメニューやショートカットで任意の場所を記録
- **サイドバーUI**: 履歴ログとユニーク訪問先の2ビューをタブで切り替え
- **ガターアイコン**: 手動マークした行にピンアイコンを表示
- **キーボードナビゲーション**: 履歴を前後に移動するショートカット
- **永続化**: ワークスペースごとに履歴を保存・復元

## 開発

```bash
npm install
npm run build
# F5 で Extension Development Host を起動
```
